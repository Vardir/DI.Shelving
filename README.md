# DI.Shelving

Simple DI container custom implementation.

## Getting Started

Download and reference NuGet package **Vardirsoft.DI.Shelving** in your probject.
For code examples see Wiki.

### Prerequisites

What things you need to install the package:

```
.NET Standard 2.1
```
